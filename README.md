# SEDS 537 Take-Home Midterm

This workspace produces a reproducible submission for the five-question machine learning take-home.

## Setup

1. Create a virtual environment: `python3 -m venv .venv`
2. Install dependencies: `.venv/bin/python -m pip install -r requirements.txt`
3. Open the notebook: `jupyter notebook notebook/midterm.ipynb`
4. Run the setup cell at the top of the notebook before the main import cell

## Deliverables

- `notebook/midterm.ipynb`: main experiment notebook
- `report/main.tex`: LaTeX report source
- `report/main.pdf`: compiled report
- `report/figures/`: figures used by the report
- `data/`: local dataset files used by the notebook
- `outputs/`: generated tables and figures from the notebook
